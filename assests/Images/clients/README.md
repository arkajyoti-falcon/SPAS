# Client Logos Directory

This directory contains client logos for the proposal automation system.

## Required Logo Files

Place the following logo files in this directory:

1. `zepto_logo.png` - Zepto client logo
2. `flipkart_logo.png` - Flipkart client logo
3. `shiprocket_logo.png` - Shiprocket client logo
4. `delhivery_logo.png` - Delhivery client logo
5. `swiggy_logo.png` - Swiggy client logo
6. `bosta_logo.png` - Bosta client logo
7. `noon_logo.png` - Noon client logo

## Logo Requirements

- **Format**: PNG, JPG, or JPEG
- **Recommended size**: 300x100 pixels (or similar aspect ratio)
- **Background**: Transparent (PNG) or white background
- **Quality**: High resolution for professional appearance

## How It Works

When you select a client from the dropdown in Section 1, their logo will automatically appear in the "Client Logo Preview" area. If you select "+ Add New Client", you can upload a custom logo.

## File Naming

Ensure logo filenames match exactly as shown above (lowercase, underscore-separated).
